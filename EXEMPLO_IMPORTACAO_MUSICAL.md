## Passos para Criar o PDF

### Usando Microsoft Word ou Google Docs:

1. Abra um novo documento
2. Insira uma tabela com 6 colunas
3. Preencha o cabeçalho: Nome | Congregação | Cidade | Telefone | Instrumento | Etapa
4. Adicione os dados de cada músico nas linhas seguintes
5. Vá em Arquivo > Salvar Como > PDF
6. Salve o arquivo

### Usando Excel ou Google Sheets:

1. Abra uma nova planilha
2. Preencha as colunas conforme o exemplo
3. Vá em Arquivo > Exportar > PDF
4. Salve o arquivo

### Usando um Editor de Texto:

1. Copie o conteúdo do exemplo acima
2. Cole em um editor de texto (Notepad, VSCode, etc.)
3. Salve como .txt
4. Use um conversor online de TXT para PDF ou imprima como PDF

## Validação dos Dados

Antes de importar, certifique-se de que:

✅ Todas as congregações mencionadas existem no sistema  
✅ Os instrumentos estão escritos exatamente como na lista válida  
✅ As etapas estão escritas exatamente como: Ensaio, RJM, Culto Oficial ou Oficialização  
✅ Os telefones estão no formato correto  
✅ Não há linhas vazias entre os registros
